How to get started

On Home page you can:

Login if you have already registered

Or

Click Register to register

 How to register
  1. Click the Register button on the Home page
  2. Select Category, Constituency, Ward then Village and click Next button
  3. On Security details
    • Choose secret question
    • Enter secret answer
    • Enter your password
 Login

To Login you must have registered!

  1. Enter your National Id Number
  2. Enter your Password
  3. Click Login button
  4. Incase you forgot password click Forgot PIN
 Forgot PIN

Click Forgot PIN

On a popup window enter your National Id Number then click Ok

 How to apply for the loan

After login being successful, you will be able to see:Apply for mobile loan, My Savings and Loans

Apply for mobile loan

The loan limit is Kshs 5,000

  • Click on Apply Loan
  • Fill all the fields on:
    • loan Application and save
    • collateral and save
    • Referees and save
